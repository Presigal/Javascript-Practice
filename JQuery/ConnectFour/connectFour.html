Connect Four!

Project by Alejandro Cespedes

Connect Four is a fun game that can be played with friends, and this website will work as an example you can play with friends!

The goal of the game is simple, have four circles that are the same color in a row, and you win. It is almost like tic tac toe, the only difference is that you have to account for the pieces falling to the bottom. It's real life counterpart is a vertical board with the pieces falling to the bottom. Best of luck!
